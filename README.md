# 🔐 Arduino Keypad Safe (PIN Access System)

Простой проект "сейфа" на Arduino с использованием матричной клавиатуры 4x4.
Пользователь вводит PIN-код, после чего система проверяет его и выводит результат в Serial Monitor.

---

## 📌 Описание

Этот проект реализует базовую систему контроля доступа:

* Ввод PIN-кода через клавиатуру
* Подтверждение ввода кнопкой `#`
* Сброс ввода кнопкой `*`
* Маскирование ввода (`*` вместо цифр)
* Проверка правильности PIN-кода

---

## 🧰 Используемые компоненты

* Arduino (Uno / Nano / любой совместимый)
* Клавиатура 4x4 (матричная)
* Провода (jumper wires)

## 🚀 Как работает

1. При запуске в Serial Monitor появляется сообщение:

   ```
   Введите PIN (# - для подтверждения, * - для сброса)
   ```

2. Пользователь вводит цифры:

   * Каждая введённая кнопка отображается как `*`

3. Нажатие `#`:

   * Проверяет введённый PIN
   * Если верный → `Access available!`
   * Если неверный → `Access not available! Try again!`

4. Нажатие `*`:

   * Сбрасывает ввод
   * Выводит `RELOAD...`

## 📜 Код

```cpp
#include <Keypad.h>

const byte ROWS = 4;
const byte COLS = 4;
char keys[ROWS][COLS] = {
  { '1', '2', '3', 'A' },
  { '4', '5', '6', 'B' },
  { '7', '8', '9', 'C' },
  { '*', '0', '#', 'D' }
};
byte rowPins[ROWS] = { 11 , 10 , 9 , 8 };
byte colPins[COLS] = { 4 , 5 , 6 , 7 };

Keypad kp = Keypad(makeKeymap(keys), rowPins, colPins, ROWS, COLS);

String input = "";
const String correctPIN = "2482";

void setup() {
  Serial.begin(9600);
  Serial.println("Введите PIN (# - для подтверждения, * - для сброса)");
}

void loop() {
  char but = kp.getKey();
  if (!but) return;

  if (but == '#') {
    if (input == correctPIN) {
      Serial.println();
      Serial.println("Access available!");
    } else {
      Serial.println();
      Serial.println("Access not available! Try again!");
    }
    input = "";
  } else if (but == '*') {
    input = "";
    Serial.println();
    Serial.println("RELOAD...");
  } else {
    input += but;
    Serial.print("*");
  }

  delay(50);
}
```
